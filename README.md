# Lumia — Deep Work Landing Page

A modern and elegant landing page for Lumia, a productivity and deep work platform.  
Built with HTML5 and CSS3, featuring premium UI sections, smooth layouts, and responsive design.

## Features
- Modern hero section
- Interactive navigation
- Feature showcase cards
- Testimonials section
- Call-to-action section
- Responsive and clean design

## Technologies Used
- HTML5
- CSS3
- Google Fonts

## Purpose
This project was created to practice advanced frontend layout design and modern landing page styling.
